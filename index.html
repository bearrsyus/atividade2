A ideia de conjuntos numéricos surge a partir da contagem que é uma relação entre dois elementos quaisquer. Os conjuntos numéricos reúnem diversos conjuntos cujos elementos são números. Eles são formados pelos números naturais, inteiros, racionais, irracionais, reais e complexos.




Conjuntos numéricos principais:
Conjunto dos Números :
Compreende os números inteiros positivos, incluindo o zero {0, 1, 2, 3,...}.
Conjunto dos Números :
Inclui os números naturais, seus opostos negativos e o zero {-3, -2, -1, 0, 1, 2, 3,...}.
Conjunto dos Números :
Formado pelos números que podem ser expressos como fração (a/b), onde a e b são inteiros e b é diferente de zero.
Conjunto dos Números :
São números que não podem ser expressos como uma fração, ou seja, sua representação decimal é infinita e não periódica.
Conjunto dos Números :
É a união de todos os números racionais e irracionais. Representa a reta numérica.

Conjunto dos Números Complexos(C):
Este conjunto é uma extensão dos números reais, incluindo a unidade imaginária (i), definida como a raiz quadrada de -1 (√-1).
Os números complexos são expressos na forma a + bi, onde a e b são números reais e i é a unidade imaginária.
Os números são um subconjunto dos números complexos.
